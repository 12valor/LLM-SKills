# Vetted Non-Generic Font Pairings

Avoid: Inter, Roboto, Arial, Space Grotesk (overused), Poppins (overused),
Montserrat (overused), Lato (overused).

All pairings below are available on Google Fonts unless noted.

---

## Editorial / Literary
- Display: **Playfair Display** (high-contrast serif, elegant)
- Body: **Source Serif 4** (readable, refined)
- Use for: Publishing, portfolio, luxury, cultural institutions

## Technical / Developer
- Display: **Syne** (geometric, slightly odd proportions)
- Body: **DM Mono** or **JetBrains Mono**
- Use for: Dev tools, terminals, technical documentation

## Humanist / Warm
- Display: **Fraunces** (optical size variable, quirky)
- Body: **Atkinson Hyperlegible** (designed for low vision — warm, practical)
- Use for: Nonprofits, accessibility-first, editorial with heart

## Industrial / Utilitarian
- Display: **Bebas Neue** (condensed, loud) or **Barlow Condensed**
- Body: **IBM Plex Sans** (engineered, neutral but not bland)
- Use for: Manufacturing, logistics, sports, hard-utility apps

## Contemporary Magazine
- Display: **Cabinet Grotesk** (variable, slightly irregular)
- Body: **Lora** (transitional serif, works small)
- Use for: Media, blogs, content-first products

## Refined / Luxury
- Display: **Cormorant Garamond** (ultra-thin, aristocratic)
- Body: **Jost** (clean grotesque, not bland)
- Use for: Fashion, jewelry, high-end hospitality

## Playful / Consumer
- Display: **Fredoka** or **Nunito** (rounded, friendly)
- Body: **Plus Jakarta Sans** (contemporary, legible)
- Use for: Consumer apps, kids adjacent, social products

## Brutalist / Raw
- Display: **Archivo Black** (heavy, condensed) or **Anton**
- Body: **Space Mono** (monospace, code-adjacent)
- Use for: Brutal design aesthetic, portfolios, experimental UIs

## Elegant Monospace
- Display: **Courier Prime** (refined Courier, editorial)
- Body: **Courier Prime** (all mono, intentional constraint)
- Use for: Writing tools, typewriter aesthetic, zine-style

## Geometric / Bauhaus
- Display: **Oswald** or **Raleway** (geometric, structured)
- Body: **Libre Franklin** (news gothic, practical)
- Use for: News, sports, structured information hierarchies

---

## Pairing Rules

1. **Contrast beats harmony**: A serif display + sans body creates interest.
   A sans display + sans body needs strong size contrast to compensate.
2. **One variable font can do a lot**: Use weight axis for hierarchy instead
   of adding a third typeface.
3. **Mono is underrated**: Using monospace for UI labels/captions (not just code)
   adds texture and precision.
4. **Never use three fonts**: Two maximum. If you need more contrast, use
   the weight/style axis of one of them.
